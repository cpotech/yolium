## What was implemented

- Reused the shared `StatusBar` in the Scheduled Agents tab instead of rendering schedule content without footer chrome.
- Generalized `StatusBar` so it can render a non-project label (`contextLabel`) without fake path or container metadata.
- Added renderer coverage for generic `StatusBar` mode and a Playwright regression spec for the Scheduled Agents footer.
